ActiveX controls (OCX)

Define

A software component that encapsulates functionality typically provided by an application and makes it available to a client application via properties, functions, and events.

Requirements

Our application that uses some ActiveX controls (.ocx or dll file), the .ocx files must be registered correctly for it to work correctly.
