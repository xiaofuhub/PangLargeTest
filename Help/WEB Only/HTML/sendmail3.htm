Send emails from practitioner

  1. Select menu File -> Practitioner. The Practitioner window displays.
  2. Click the Image tab, select an image to send, and click the Email menu. The Email window displays, as shown below.
  3. Input relevant information and the email body.
  4. Click the Send button to send emails.The image will be sent as an attachment.
