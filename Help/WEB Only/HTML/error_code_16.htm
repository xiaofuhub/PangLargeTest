Intellisoft/Email send/ Error code 16

Error code 16

Problem:Error Id=16

There was a problem with the connection and a socket error occurred.

Solution:

    • Check that the SMTP server is started.
    • Check that the address and port of the SMTP server are valid.
      • Check that the server address is typed correctly and the format is correct. The server address is not in block list of any other software.
      • Check that the port is opened (use the telnet command to test, for example, telnet 192.0.0.9 25) and that the virus scanner, proxy server, and firewall allow the port of SMTP protocol.
    • Check that the sender¡¯s email address is typed correctly and the format is correct.
